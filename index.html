Audible Projects (Non-Confidential)

Project 1: Creating More Examples

LLMs require examples in given prompts to perform desired actions. This is a prompting technique known as few-shot prompting. We have several examples created already but the model may improve when given more examples. You will create more examples to add into the prompt and then analyze if there was any improvement in character detection.

Project 2: Optimizing Multicast for Prompt Caching

The Prompt Caching Project focuses on optimizing the upcoming Bedrock feature that enables prompt caching. This feature will allow for more efficient processing by storing and reusing prompts. To fully leverage this capability, we plan to replace the current approach of summarizing content with including the entire chapter directly within the prompt. This change is expected to enhance performance and reduce processing time, leading to a more streamlined and efficient user experience.

Project 3: Using DeBERTa for Character Disambiguation

This project aims to address inconsistent character labeling in Virtual Voice Audiobooks (VVABs). We're exploring two methods: (1) enhancing LLM prompts to disambiguate character labels, and (2) using a Bidirectional Entailment Clustering Algorithm (DeBERTa) to detect semantically similar labels. A proof of concept showed promising results in identifying label relationships. Next, we'll create a ground truth dataset to evaluate these methods' accuracy and performance before deciding on the best approach.
